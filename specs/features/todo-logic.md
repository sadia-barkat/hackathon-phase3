# Feature: Todo Core Logic
## Requirements
The application must allow users to:
1. **Add Task:** Input a string and save it as a task.
2. **View Tasks:** List all tasks with an index.
3. **Update Task:** Edit the text of an existing task.
4. **Delete Task:** Remove a task from the list.
5. **Complete Task:** Mark a task as done.

## Technical Constraints (Phase I)
- Language: Python
- Interface: Command Line (Console)
- Persistence: In-memory (List)
